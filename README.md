Projeto acadêmico inspirado no universo de **Berserk**, desenvolvido como uma aplicação SPA com padrão MVC.

## 🌐 Aplicação online

Adicione o link da aplicação publicada aqui:

```text
https://berserk-archive.netlify.app
```

## ✨ Funcionalidades

- Página inicial e página sobre
- Catálogo de produtos e serviços
- Pesquisa de itens
- Cadastro e login de usuários
- Painel administrativo
- Cadastro, edição e exclusão de registros
- Formulário de contato
- Integração com Supabase
- Layout responsivo

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript
- Supabase
- Netlify
- SPA
- MVC
